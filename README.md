# ⏳ Loader Animado em CSS

Um projeto simples, elegante e totalmente responsivo que apresenta um **loader animado**, criado com HTML e CSS puro.  
O objetivo é demonstrar como microinterações podem transformar a experiência do usuário, deixando interfaces mais vivas e profissionais.

---

## ✨ Destaques

- 🎨 Animação suave e moderna  
- 💡 Criado apenas com **HTML + CSS**  
- 📱 Totalmente responsivo  
- ⚡ Ideal para splash screens, telas de carregamento e projetos web  
- 🧩 Código limpo e fácil de adaptar  

---

## 🛠 Tecnologias utilizadas

- **HTML5**
- **CSS3 (keyframes, transform, animation)**
